Simple Usage of Equal Height Plugin

This div has Height = 300px

Height takes the exact height of the content which excludes Padding, Border & Margin

There is no height applied to this div.

Plugin is setting its height equal to the leftdiv's height

Different types of Height can be taken using plugins options

This div has Height = 300px

InnerHeight takes the height of the content + padding excluding the Border and Margin

There is no height applied to this div.

Plugin is setting its height equal to the leftdiv's innerHeight

This div has Height = 300px

OuterHeight takes the height of the content + padding + border and Excludes the Margin

There is no height applied to this div.

Plugin is setting its height equal to the leftdiv's outerHeight
